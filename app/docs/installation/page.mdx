# Installation

## Prerequisites

- Node.js 18+
- Next.js 14+ (App Router)
- Tailwind CSS 3.4+

## Install the CLI

```bash
npm install -g nyuchi@latest
```

## Initialize in your project

```bash
npx nyuchi@latest init
```

This will:
1. Create `components.json` configuration
2. Set up the `@/lib/utils` helper with `cn()` function
3. Configure Tailwind with Nyuchi design tokens
4. Install the Five African Minerals color palette
5. Set up the Noto Serif / Noto Sans / JetBrains Mono font stack

## Add components

```bash
# Add individual components
npx nyuchi@latest add button card dialog

# Add a full layer
npx nyuchi@latest add --layer 3

# Add all components for a mini-app
npx nyuchi@latest add --mini-app wallet
```

## Project structure

```
src/
├── components/
│   ├── ui/          # Layer 2 — Primitives
│   ├── brand/       # Layer 3 — Brand Components
│   ├── safety/      # Layer 4 — Safety Gates
│   ├── resilience/  # Layer 5 — Resilience
│   ├── pages/       # Layer 6 — Page Compositions
│   └── shell/       # Layer 7 — App Shell
├── lib/
│   ├── tokens/      # Layer 1 — Design Tokens
│   ├── icons/       # Layer 1 — Icon Registry
│   ├── harness/     # Harness — Cross-cutting
│   ├── assurance/   # Layer 8 — Assurance
│   └── fundi/       # Layer 9 — Fundi
└── docs/            # Layer 10 — Documentation
```
