# Pipeline

## `GET /api/v1/pipeline`

Returns the open data pipeline stages for the Mukoko ecosystem. The pipeline flows from event ingestion through processing to analytics: **Redpanda** (ingestion) to **Apache Flink** (processing) to **Apache Doris** (analytics).

### Response Shape

| Field | Type | Description |
|---|---|---|
| `stages` | `array` | Ordered pipeline stages |

### Example Stage

```json
{
  "name": "ingest",
  "technology": "Redpanda",
  "description": "Kafka-compatible event streaming. Ingests change events from CouchDB and Supabase.",
  "status": "planned"
}
```

```json
{
  "name": "process",
  "technology": "Apache Flink",
  "description": "Stream processing — strip PII, enrich, aggregate.",
  "status": "planned"
}
```

```json
{
  "name": "serve",
  "technology": "Apache Doris",
  "description": "Real-time OLAP. Powers open data API and dashboards.",
  "status": "planned"
}
```

### Example

```bash
curl https://registry.mukoko.com/api/v1/pipeline
```
